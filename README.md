# DashBoard-Nutricional
Este projeto consiste na construção de um dashboard interativo em Power BI para análise nutricional de produtos de grandes redes de fast food.  O objetivo foi transformar dados brutos em insights estratégicos, avaliando o teor de açúcar, sódio e calorias dos produtos com base nas recomendações oficiais da Organização Mundial da Saúde (OMS).
